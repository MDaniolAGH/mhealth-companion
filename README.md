# Mobile Apps for Healthcare — Starter Code

Non-graded lab starter projects for the *Mobile Apps for Healthcare*
course at AGH University of Kraków (Biomedical Engineering, 3rd year).

**Full course materials** (lecture notes, lab workbooks, cheatsheets,
quizzes) live on the course site, behind the shared course password:
<https://mdaniolagh.github.io/mhealth/>

## How to use

Clone once at the start of the course:

```bash
git clone https://github.com/MDaniolAGH/mhealth-companion.git
cd mhealth-companion
```

Pull updates every week and after any mid-week fix:

```bash
git pull
```

Each week has one or more starter projects in a numbered directory.
For Flutter labs, open the project folder in your IDE and run:

```bash
flutter pub get
flutter run
```

For the Week 2 FastAPI lab, follow the week's workbook for setup.
